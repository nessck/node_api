# Nodejs API
Jr. Devops Engineer Case Study for Codeway Studios

## Tech

-   Nodejs
-   Express and Postgres
-   Docker
-   Docker Compose
-   Kubernetes
-   Helm
-   Minikube

## Prerequisites

-   Docker on local machine
-   Minikube

## Run Locally

Clone the project

```bash
  git clone **git url**
```

Go to the project directory

```bash
  cd Node_API
```

> *Before next step please make sure you have Docker installed on local machine*

Run docker compose


```bash
  docker-compose up -d
```

Check docker containers with

```bash
  docker-compose ps
```

Stop all containers of the project

```bash
  docker-compose down
```
